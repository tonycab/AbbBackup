PC SDK

PC SDK allows system integrators, third parties or end-users to add their own customized operator interfaces for an ABB robot controller. It can also be used to develop RobotStudio add-ins that interacts with the controller. With PC SDK it is possible to create an application that connects to one or several IRC5 or OmniCore controllers, real as well as virtual.
