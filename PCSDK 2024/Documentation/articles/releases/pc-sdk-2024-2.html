Release Notes for PC SDK 2024.2

General

The release name is PC SDK 2024.2 and the build number is 24.2.10789.0. The build date is May 23, 2024.

Corrections

PDD
GetAllMessages method - Anomalies in Event Retrieval

EventLog.GetAllMessages previously ignored the maxNumberOfEvents parameter. It no longer does. One call will not return more than 1000 messages but consecutive calls can be used to get more.
